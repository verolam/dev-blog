JavaScript Array

What is a JavaScript Array ?

An array is a special variable, which can hold more than one value at a time.

Step by step description

First create an array (in our example named "todoList") :

              
                let todoList = [
                "Go shopping",
                "Buy tobacco",
                "Do this",
                "Do that"
                ];

Access to the element of an Array:
We access to one element of an array with an index.
Note: Array indexes start with 0. 

              
                let elt = todoList[0]; //elt will have as value "Go shopping";

You can add an element to the ARRAY.

              
                todoList.push("Do also that (it important");

You can delete an element from the ARRAY.

              
                todoList.splice(1, 1);

To see the result you use the console.

              
                console.log("List after push",todoList);
